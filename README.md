# GTA VI - Fan-Made Website Concept

This project is a visually rich, animated, and responsive fan-made website concept for the game Grand Theft Auto VI. It was built using modern web technologies like React and GSAP to create an immersive user experience.

## ‚ú® Features

-   **GSAP Intro Animation:** An engaging splash screen animation using SVG masks.
-   **Interactive Mouse Effects:** The main title subtly follows the user's mouse movement for a dynamic effect.
-   **Responsive Design:** A mobile-first approach ensuring the site looks great on all devices.
-   **Modern Tech Stack:** Built with Vite for a fast and efficient development experience.

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

-   [Node.js](https://nodejs.org/en/) (v18 or newer recommended)
-   [npm](https://www.npmjs.com/)

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Navigate into the project directory:**
    ```sh
    cd gta-6
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```

### Running the Application

To run the app in a development environment, use the following command. This will start the Vite development server, typically on `http://localhost:5173`.

```sh
npm run dev
```
